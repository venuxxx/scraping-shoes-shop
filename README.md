Программа парсит данные с сайта https://sneakerstore.by/muzhskie-krossovki/. Папка modules представляет модули для записи спаршенных данных в разные форматы (json, xlsx, csv). Папка result scraping содержит в себе результаты парсинга данныхю. Файл scraping.py представляет собой реализацию парсера. Подключаются библиотеки requests, BeatufulSoup - для отправки запроса на сайт и обработки полученного html кода. Библиотеки jsonmodule, csvmodule, excelmodule - собственне модули для сбора данных в разные форматы. Фактором отбора для этого парсера была цена кроссовок (менее 150 руб). Парсер реализован при помощи функции-генератора, без использования примитивной записи данных в список, что делает код более оптимизированным.
